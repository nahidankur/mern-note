# A MERN Note Taking APP

Technology Used: `ReactJs`, `NodeJs`, `MongoDB`, `ExpressJs`, `HTML`, `CSS`

# [Live Preview Here](https://wild-blue-tadpole-boot.cyclic.app)

## Usage

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```
